⚙️ Configuración del tablero

Corfix proxy — Instalá la extensión CORS Unblock en Chrome (gratuita) o usá el proxy público corsproxy.io. Esto permite que el tablero llame a Jira directamente desde el browser.

Generalo en: id.atlassian.com → API tokens

Si usás extensión CORS Unblock, elegí "Sin proxy". Si no, elegí un proxy público.

Sin instalación — Ejecutá este comando en tu terminal y pegá el resultado aquí. Funciona siempre, sin CORS ni extensiones.
curl -u "TU_EMAIL:TU_API_TOKEN" \ "https://waynimovil.atlassian.net/rest/api/3/search?jql=project%3DWT%20AND%20labels%3Dsoporte%20ORDER%20BY%20created%20DESC&maxResults=100&fields=summary,status,priority,created,updated,assignee,reporter,comment,resolutiondate" \ -H "Accept: application/json" > jira_issues.json
Servidor local con Node.js — El proxy corre en tu máquina, resuelve CORS y el tablero se actualiza solo. Ideal para uso continuo en la oficina.
# 1. Instalá el proxy (una sola vez)
npm install -g local-cors-proxy

# 2. Inicialo (dejalo corriendo)
lcp --proxyUrl https://waynimovil.atlassian.net --port 8010

# 3. Configurá el tablero con localhost abajo

El proxy redirige al dominio de Jira. Dejá el servidor corriendo mientras usás el tablero.
